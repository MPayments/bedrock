# Bedrock Framework

Добро пожаловать в документацию Bedrock.

Bedrock - это финансовый framework на базе монорепозитория (Turborepo), в котором есть готовые блоки для:

- документо-ориентированных бизнес-процессов
- платежей и маршрутизации провайдеров
- бухгалтерских проводок и ledger
- фоновой обработки (workers)
- модульного управления компонентами

2. [Структура репозитория](/getting-started/repository-structure)

## Что важно понять в первые 10 минут

- **API принимает команды**, но фактическая внешняя обработка часто асинхронная.
- **Документы** (`documents`) - источник workflow-состояния.
- **Платежи** используют `connectors + orchestration` для выбора провайдера и ретраев.
- **Модульный рантайм** может включать/отключать компоненты без перекомпиляции.

## Минимальный путь до запуска

```bash
docker compose -f infra/docker-compose.yml up -d
bun install
bun run --filter=api dev
bun run --cwd apps/web dev
bun run --cwd apps/docs dev
```

После запуска:

- Web UI: `http://localhost:3001`
- API: `http://localhost:3002`
- Docs: `http://localhost:3003`
- API reference (Scalar): `http://localhost:3002/docs`
