# @multihansa/eslint-config

Shared ESLint configuration package for the monorepo.

## Exports

- `@multihansa/eslint-config/base`
- `@multihansa/eslint-config/next-js`
- `@multihansa/eslint-config/react-internal`

## Purpose

- Keep linting rules consistent across apps and packages.
- Avoid duplicating ESLint setup in each workspace.

## Notes

- This package provides config modules only.
- It has no runtime behavior.
